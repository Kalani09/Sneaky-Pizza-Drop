<h2 align="center">🍕 Sneaky Pizza Drop 🎮</h2>

<p align="center">
Sneaky Pizza Drop is a fun, fast-paced casual game where you drop pizzas with perfect timing to score points.  
Watch out for surprises, obstacles, and the occasional random chaos that keeps each round exciting!  
</p>

I built this game for **EXE – Sri Lanka’s Flagship Game Development Competition**, as my very first published game.
